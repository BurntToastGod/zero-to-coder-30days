## 🧠 How to Use This Repo

### Option A — Follow the Daily Grind (Recommended)
1. Start at [`daily-grind/day-01.md`](./daily-grind/day-01.md)
2. Do ONE day per day — don't skip ahead
3. Every day has: concept → example → predict → code → break → fix → reflect
4. Use the `roadmap/` folder to understand the big picture
5. Do exercises in `exercises/` when you want more practice

### Option B — Fork & Customize
1. Fork this repo to your own GitHub
2. Check off tasks as you complete them
3. Commit your code solutions to YOUR fork — build a portfolio as you learn!

## 🔁 The Daily Learning Loop

Every single day follows this exact loop — this is what makes it stick:

```
1. 🧠 LEARN    — Read the concept (plain English, no jargon)
2. 👁️ SEE      — Look at a tiny code example
3. 🔍 PREDICT  — Guess the output BEFORE running it
4. 💻 CODE     — Type it yourself (don't copy-paste!)
5. 💥 BREAK    — Intentionally break the code
6. 🔧 FIX      — Fix it and understand WHY it broke
7. 🧩 REFLECT  — Answer 1 question about what you learned
```

## ⚙️ Setup — What You Need

You only need **Python 3** — it's free.

**Option 1 — Online (zero install):**
- Go to [replit.com](https://replit.com) → Create a Python Repl → Start coding

**Option 2 — Local install:**
1. Download Python: [python.org/downloads](https://python.org/downloads)
2. Install VS Code: [code.visualstudio.com](https://code.visualstudio.com)
3. Open terminal → type `python3 --version` to verify
