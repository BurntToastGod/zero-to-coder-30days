# ⚙️ Functions Practice

Use after: Day 17-22

## Exercise 1: Simple Functions (Beginner)
Write a function called `greet` that takes a name and returns:
`"Hello, [name]! Welcome to Python."`

Test it with 3 different names.

---

## Exercise 2: Calculator Functions (Beginner)
Write 4 separate functions: `add`, `subtract`, `multiply`, `divide`.
Each takes 2 numbers and returns the result.
`divide` should return an error message if dividing by zero.

---

## Exercise 3: Grade Calculator (Intermediate)
Write a function `get_grade(score)` that:
- Returns "A" for 90-100
- Returns "B" for 80-89
- Returns "C" for 70-79
- Returns "D" for 60-69
- Returns "F" for below 60

Test it with scores: 95, 82, 71, 65, 45

---

## Exercise 4: List Functions (Intermediate)
Write these functions:
- `find_max(numbers)` - returns the largest number WITHOUT using built-in max()
- `find_min(numbers)` - returns the smallest number WITHOUT using built-in min()
- `calculate_average(numbers)` - returns the average

Test with: [5, 2, 8, 1, 9, 3, 7]

---

## Exercise 5: String Functions (Hard)
Write a function `analyze_text(text)` that returns a dictionary with:
- `word_count`: number of words
- `char_count`: number of characters (no spaces)
- `longest_word`: the longest word
- `uppercase_count`: number of uppercase letters

No hints. Think about what Python string methods exist.
