```python

b = "world"

# b should be "hello"
```

## Exercise 5: Temperature Converter (Hard)
Write a program that:
1. Asks for temperature and unit (C or F)
2. Converts to the OTHER unit
3. Prints the result formatted to 1 decimal place

Formulas:
- C to F: (C * 9/5) + 32
- F to C: (F - 32) * 5/9

No hints. You've got this.
