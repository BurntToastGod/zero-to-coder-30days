# 🧠 Exercises

Extra practice problems to reinforce what you learn each day.
These are OPTIONAL but highly recommended.

## How to Use
- Try each exercise WITHOUT looking at the solution first
- If stuck for 15+ minutes, check the hints
- If still stuck, look at the solution, understand it, then write it from memory

## Files

| File | Topic | When to Use |
|------|-------|-------------|
| `variables-practice.md` | Variables & Data Types | After Day 1-5 |
| `loops-practice.md` | For/While Loops | After Day 11-15 |
| `functions-practice.md` | Functions | After Day 17-22 |
| `debugging-challenges.md` | Debugging | After Day 25 |

## Difficulty Levels
- 🌱 Beginner - copy and modify
- 🔥 Intermediate - write from scratch with hints  
- 💀 Hard - no hints, just the problem

## Remember
> The point isn't to finish fast. The point is to UNDERSTAND.
> Slow and understood > fast and forgotten.
