# 🛠️ Tools & Setup Guide

Everything you need to write and run Python code.

---

## 👀 Where to Run Python RIGHT NOW (No Install)

### Option 1: Replit (Easiest)
- Go to: https://replit.com
- Create free account
- Click "+ Create Repl" -> Python
- Write code, click Run
- Great for: starting out, sharing code

### Option 2: Google Colab
- Go to: https://colab.research.google.com
- Free, no install, saves to Google Drive
- Great for: Python + data projects

### Option 3: GitHub Codespaces
- Go to your GitHub repo
- Press `.` to open in browser editor
- Or use Codespaces tab for a full terminal
- Great for: working directly in your repos

---

## 💻 Installing Python Locally (Recommended Eventually)

### Windows
1. Go to https://python.org/downloads
2. Download latest Python 3.x
3. Run installer, CHECK "Add Python to PATH"
4. Open Command Prompt, type `python --version`

### Mac
1. Go to https://python.org/downloads
2. Download latest Python 3.x
3. Run installer
4. Open Terminal, type `python3 --version`

## 📝 Best Free Code Editors

### VS Code (Recommended)
- Free, fast, great Python support
- Download: https://code.visualstudio.com
- Install "Python" extension by Microsoft
- How to run: open terminal in VS Code, type `python filename.py`

### PyCharm Community Edition
- Built for Python specifically
- Download: https://www.jetbrains.com/pycharm/
- Heavier than VS Code but very powerful

### Thonny
- Designed for beginners
- Great debugger for learning
- Download: https://thonny.org

## 🐍 Running Python Files

```bash
# In terminal/command prompt:
python filename.py       # Windows
python3 filename.py      # Mac/Linux

# Interactive mode:
python                   # Opens Python REPL
>>> 2 + 2                # Type code directly
4
>>> exit()               # Exit
```

## 📦 Installing Packages (Later)

## 🐛 When You Hit an Error
1. Read the ENTIRE error message
2. Note the line number
3. Copy the last line of the error
4. Paste it into Google or ChatGPT
5. Understand the fix, don't just copy it
