## 🔥 Your Final Challenge: BUILD SOMETHING NEW

This is the graduation test. No instructions. No code to copy.
Just you and Python.

Choose ONE of these:

### Option 1: Number Guessing Game
The computer picks a random number between 1-100.
The player guesses. The computer says "higher" or "lower".
Track how many guesses it took.
Show a rating at the end (Amazing/Good/Could be better).

### Option 2: Password Generator
Ask the user how long they want the password.
Ask if they want: uppercase, numbers, symbols.
Generate and display a random password.

### Option 3: Todo List App
User can: add task, view tasks, mark complete, delete task.
Show the list after each action.
Tasks stay in a list until deleted.

### Option 4: YOUR IDEA
Build something YOU want to exist.
It must use: functions, a loop, user input, and a list or dictionary.
